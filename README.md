# FilePro

FilePro is a patient records application focused on simple, day-to-day data operations.

## Functionality

- User login with username-based session behavior.
- New user signup flow.
- Dashboard with patient management view.
- Upload patient data using Excel (`.xlsx`) files.
- View patient records in a tabular list.
- User-facing error messages when patient loading, signup, or upload actions fail.

## Dependencies

- Node.js (18+ recommended)
- npm
- React
- React DOM
